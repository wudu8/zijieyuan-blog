# 电饭锅一锅出

米饭 + 菌菇 + 火锅菌菇汤料 + 三文鱼（非必选）--30min

米饭（酱油、黄油）30min 煮熟 + 牛肉（酱油、蚝油、沙茶酱、糖、面粉腌制 ）+ 冷冻蔬菜（玉米，豌豆，胡萝卜）10min 煮熟 -- 40min

茄子、番茄（切小块打底，不需要水）+ 香肠（非必选）+ 顶部铺鲜面条【出锅加酱油、香油、蒜蓉、葱花、醋】 --30min

番茄 + 香肠 + 海底捞番茄酱汤包 + 通心面 --30min

土豆 + 胡萝卜 + 洋葱 + 去骨鸡腿肉 + 咖喱块 + 最后 10min 加入乌冬面 --40-50min 煮烂土豆

番茄锅底 + 油面筋（内部可以注入虾滑）+ 起锅前 15min 加入粉丝 --30min

白菜切烂打底 + 粉丝 + 虾仁 + 蛤蜊 + 生抽、老抽、耗油、料酒、麻椒油 【注意不用放水】起锅加蒜蓉、小米辣调味 --30min

油打底 + 排骨 + 土豆 + （老抽、生抽、蚝油、蜂蜜、料酒、黄豆酱、八角）（30min 煮烂土豆） + 最后 20min 加入豆角 + 面饼 --50min

火锅底料 + 米 + 丸子、肥牛、火腿 --材料熟透

生姜、洋葱、大蒜 + 鸡腿 + （生抽老抽耗油糖胡椒粉）注意不用加水 + 起锅前 10min 加入花卷 --30min

酸菜、肉末、蒜蓉 10min + 水 + 番茄 + 木耳+ 豆芽 + 米线 （酱油、味精、白胡椒）15min --25min

牛腩（浸泡 30min 去除血水）+ 番茄 + 土豆 （姜片、酱油、蚝油、八角、糖、桂皮）+ 出锅前 10min 加入面当主食 --1.5h

排骨（浸泡 30min 去除血水）+ （生抽、老抽、陈醋、冰糖、姜片、山楂｜乌梅、可乐）+ 起锅前加入宽粉当主食--1h
